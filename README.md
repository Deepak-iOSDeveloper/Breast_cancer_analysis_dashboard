## Project Overview
Interactive Power BI dashboard analyzing 12,000 breast cancer 
patient records from the SEER (Surveillance, Epidemiology, 
and End Results) Program.

## Dataset
- **Source:** SEER Cancer Registry Program
- **Records:** 12,000 patients
- **Columns:** 16 clinical variables

## Dashboard Features
- 6 interactive charts (Donut, Bar, Column, Pie, Stacked, Grouped)
- Dynamic metric switcher
- Age range slider filter
- Cross-filtering across all charts
- Multi-page navigation
- Dark sidebar with checkbox slicers
- KPI cards with conditional formatting

## Charts Included
1. Survival Status — Donut Chart
2. Patients by T Stage — Horizontal Bar
3. Avg Survival by Grade — Column Chart
4. Hormone Receptor Status — Grouped Bar
5. N Stage Breakdown — Stacked Column
6. Race Distribution — Pie Chart

## How to Open
1. Download `BreastCancer_Dashboard_Deepak.pbix`
